# Autres
Contient différents autres outils. / Contains various other tools. (ArcGis 10.5 / ArcMap / Add-in / Exe / Vb.Net)

BarreNonConformite (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)
-Barre de menu utilisée dans dans ArcMap pour gérer les non-conformités des données BDG et RHN et gérer les réplicas pour la correction des données (CheckOut) à partir d'une Géodatabase SDE. 
 Pour plus de détails, voir le fichier de documentation "BarreNonConformite.doc" présent dans ce dépôt.
-Menu bar used in in ArcMap to manage nonconformities of BDG and RHN data and manage data replicas for data correction (CheckOut) from an SDE geodatabase.
 For more details, see the documentation file "BarreNonConformite.doc" present in this repository.

-Barre de menu utilisée dans dans ArcMap pour gérer l'affichage de l'information sur les différentes bases de données et différents produits. 
 Pour plus de détails, voir le fichier de documentation "BarreSib.doc" présent dans ce dépôt.
-Menu bar used in in ArcMap to manage the display of information on different databases and different products.
